Column Descriptions:
Data_Name - Data set unique identifier
Data_Type - Data type identifier (e.g., in vivo, in vitro, in silico, measured)
Data_Version - Data version identifier
Record ID - Unique number identifying the record
Chemical_Name - Name of test substance
CASRN - Chemical Abstract Registry Number
DTXSID - DSSTox substance identifier (unique substance identifier)
InChiKey - Chemical structure identifier for a chemical based on the chemical's International Chemical Identifier
SMILES - Simplified Molecular Input Line Entry Sytem
Kingdom - First level of chemical hierarchy
Superclass - Second level of chemical hierarchy
Class - Third level of chemical hierarchy
Subclass - Fourth level of chemical hierarchy
Level_5 - Fifth level of chemical hierarchy
Level_6 - Sixth level of chemical hierarchy
Level_7 - Seventh level of chemical hierarchy
Level_8 - Eigth level of chemical hierarchy
Level_9 - Nineth level of chemical hierarchy
Level_10 - Tenth level of chemical hierarchy
Structure_Used - Structure used to obtain ClassyFire Classification (either InChiKey or Smiles)
Tool_Used - ClassyFire tool used to obtain ClassyFire Classification (either API or base web tool)
Date_Extracted - Month that chemical classifications were obtained
Reference - Data reference
URL - Reference DOI or URL
